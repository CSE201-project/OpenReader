        Connexion au device Solarium

* Lancer l'application Solarium Workbench

* Rentrer un nom d'hôte valide dans le champ 'Device IP'
    - si on saisie un nom invalide, le bouton de connexion est désactivé
    - voici des exemples valides :
        - localhost
        - example.com
        - foo.example.com
        - bar.foo.example.com
        - exa-mple.co.uk
        - xn--80ak6aa92e.com
        - 9gag.com
        - 8.8.8.8
        - 127.0.0.1

* Cliquer sur le boutton 'Connect': 
    - Si la connexion réussie, le boutton devient vert et le nom du device s'affiche à l'écran
    - Si la connexion échoue, le chargement s'arrête au bout de 3 secondes
